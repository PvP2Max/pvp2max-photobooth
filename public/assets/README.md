# BoothOS Assets

Place branding files here for the web app:
- `logo.svg` / `logo.png` – main lockup shown in marketing/headers.
- `watermark.png` – optional watermark for free-tier renders (replaces the default BoothOS mark). Use transparent PNG or SVG at least 1600px wide.
- Overlay art or misc. brand images as needed.

Files in this folder are served statically via `/assets/<filename>` (e.g., `/assets/logo.svg`).
